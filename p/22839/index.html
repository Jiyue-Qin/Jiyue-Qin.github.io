整合一些IDEA使用技巧

发表于 2020-07-16 分类于 工具 热度： ℃ Valine： 本文字数： 1k 阅读时长 ≈ 1 分钟

这里整理了一些比较实用的IDEA使用快捷键与小技巧，后续会随时补充。。。

1. 前进/后退

使用浏览器时可以点击←，→箭头来前进后退，IDEA也有相似功能，如果在使用IDEA编辑时，查看代码逻辑或跳转到其他类想要返回时，可以用该功能快速定位之前的编辑位置（甚至DeBug时也可以用）。

前进/后退快捷键如下：

  • Ctrl + Alt + left/right（方向键左/右）

2. 查看历史记录

IDEA默认可以打开10个窗口查看文件，超过的会按照打开的顺序关闭。

如果打开标签页过多，超过一屏之后，IDEA就会把标签页收纳起来，再次使用时需要点击按钮才会显示被隐藏的标签页。

这时使用查看历史记录功能就会方便很多，它可以让我们快速的定位到想要编辑的历史页面上。

快捷键如下：

  • Ctrl + E

此外，IDEA还可以查看最近修改代码的位置，快捷键如下：

  • Ctrl + Shift + E

如果想修改标签限制，可以在设置中更改：setting → Editor → Editor Tabs →Tab Limits

3. 自动代码补全

使用这个快捷键可以自动补全代码末尾的分号，括号对等等，还可以补全少打的字符。

快捷键如下：

  • Ctrl + Shift + Enter

这个功能非常实用，可以在其他地方也试试，可能还有妙用。

同时，Shift + Enter 可以直接在当前行下增加一个空白行，编辑时同样能提高编码效率。

5. 重命名

可以更改变量名（同时改动相关引用的地方）

快捷键如下：

  • Shift + F6

该快捷键同时适用于给项目中的文件重命名

6. 快速跳转到错误行

  • Shift + F2

7. 快速生成迭代循环

  • itco + Enter

8. 快速生成List遍历方法

  • itli + Enter

9. 快速生成for each循环

  • iter + Enter

10. 快速切换字段大小写

  • Ctrl + Shift + U

11. 快速删除一行

  • Ctrl + Y

12. 快速进入方法或实现类

  • Ctrl + Alt + B

13. 添加循环体

  • Ctrl + Alt + T

14. 万能的插入快捷键

  • Alt + insert

该方法适用于插入类，插入方法，循环体等等
